Chat application client UI made on Angular

- Send message
- Reply to message
- Forward message to any chat
- Edit own message
- Delete message
- View profiles

- Chats search
- Global messages search
- Local messages search (in certain chat only)
- Jumping to messages (found, replies)
- Jumping to date
